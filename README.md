# Char Dashboard

Frontend dashboard for the Char Engine AI SaaS prototype.

## Features

- Login UI
- JWT token storage
- Protected AI dashboard
- AI prompt interface
- Conversation history display
- Tailwind-styled SaaS layout

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- Axios

npm run dev
